PyPI Mirrors Health Status
===========================

## 缘起
由于众所周知的原因， 国内访问 PyPI 官方源的速度很不稳定。 尽管有众多的第三方源，但这些第三方源的稳定性欠佳， 都曾经出现长时间未和官方源同步的问题。 本项目会每天检查各第三方源的同步状态， 便于大家评估和选择靠谱的源。
如果想知道各个源同步问题的历史记录，可以查看项目里的 outages.txt 文件
